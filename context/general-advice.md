# EXPLAINING CONTEXT TO THE AI

If you're using an IDE or even a chat interface, it's important to clearly provide context to the AI to increase the chances of getting accurate and high-quality results.

> **Note:** In an IDE, you may need less explanation, but it's still necessary to understand part of this context before asking for something.

In general, do the following:

1. Briefly explain what you want in a few lines.
2. Show the relevant files and/or code snippets.
3. If you're not sure which parts are relevant, ask the AI whether something is missing.

> ⚠️ **Caution:** The AI may misinterpret things if context is incomplete and draw incorrect conclusions.  
> **Make it explicit that it should make decisions**, and if it's uncertain about any part, ask it to point that out so you can provide the missing context.
